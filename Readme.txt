Steps to install COUNTRY CAPITAL QUIZ:

1. Clone the repository on your local system (https://github.com/MSShiraj/CountryCapitalQuiz.git)
2. Install all the required packages to a virtual environment from requirement.txt by using the following command:
        > pip install -r requirement.txt
3. To run the project, open command prompt from terminal and use  the following command:
        > python manage.py runserver
4. Now you will get into the home page that contains the name of a country and you'll have to enter
   the capital of that country in the input box provided.
5. If your answer is correct, you'll see the capital's name in 'Green' otherwise 'Red'.
